This is a LALR (1) syntax analysis generator - Mathew.

这是一个LALR(1)型语法分析生成器―Mathew

Mathew is a small automatic syntax analysis generating tool. No matter what platform programming language you use, you only need to write a driver template file that belongs to your platform language according to the driver template file specifications provided by Mathew. When you do not need to recompile Mathew, you can automatically generate the syntax rules. The syntax analysis of the platform drives the file.
Mathew inherits many ideas and functions of many LEMON, and is very grateful to the author of LEMON Richard Hipp for her selfless dedication.
  

Mathew是一个小巧的语法自动分析生成工具，无论你使用什么平台的编程语言，只需要按照Mathew提供的驱动模板文件规格编写出属于自己平台语言的驱动模板文件，在不需要重编译Mathew的情况下输入语法规则就可以自动生成相应平台的语法分析驱动文件。
Mathew继承了许多LEMON的许多思想和功能，在这里非常感谢LEMON的作者Richard Hipp无私奉献的精神。

It is hoped that Matthew can be used to develop a special computer language Mahoro.（希望马修能用于开发一款特殊的计算机语言Mahoro）
